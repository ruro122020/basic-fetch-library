# Frontend CRUD Fetch Library

This is a library with CRUD asyn functions to use with vite+react apps when communicating with the flask-postgresql-template found [here](https://github.com/ruro122020/flask-postgresql-template-local).

*url: The endpoint (Example: '/api/{endpoint})

*body: The object with data being sent to the api

### Methods
1. retrieve(url)
2. send(url, body)
3. alter(url, body)
4. remove(url)
